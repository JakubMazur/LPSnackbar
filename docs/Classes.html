The following classes are available globally.

  • LPSnackbar

    The controller for an LPSnackbarView.

    This class handles everything that has to do with showing, dismissing and performing actions in a LPSnackbarView. There are several static helper methods, which allow presenting a basic snack without needing instantiate an LPSnackbar yourself.

  • LPSnackbarView

    The LPSnackbarView which contains 3 subviews.

    • titleLabel: The label on the left hand side of the view used to display text.

    • button: The button on the right hand side of the view which allows an action to be performed.

    • seperator: A small view which adds an accent that seperates the titleLabel and the button.
